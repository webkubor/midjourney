# 人物多角度影棚效果

关于人像,需要生成的图片角度,镜头放在哪里拍摄很重要,包括但不限于人像相关的图

如果需要图片能精准的按照指示生成,需要提高其优先级和用重复的同义词

以下图为例,低角度仰视+ 极端角度从下方拍摄 + 索尼a7r4拍摄的照片


为了让图片有照片的质感,一般会加入一些常规关键词

::: info

最高品质、逼真摄影、5550 K、超分辨率、128K UHD、超逼真

:::
